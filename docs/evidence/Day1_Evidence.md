## 遇到的问题

### 问题1：pnpm命令不存在
**现象**：终端提示 `pnpm: command not found`  
**原因**：系统未安装pnpm或环境变量配置问题  
**解决**：改用npm命令替代pnpm

### 问题2：Mock服务器启动失败
**现象**：`Error: Cannot find package 'json-server\index.js'`  
**原因**：自定义server.js中json-server配置不正确  
**解决**：放弃自定义server，改用JSON Server CLI直接启动

### 问题3：API请求返回404
**现象**：前端请求 `/api/users` 返回404  
**原因**：Vite代理配置将请求转发到 `http://localhost:3000/api/users`，但JSON Server不需要`/api`前缀  
**解决**：在Vite配置中添加rewrite规则移除`/api`前缀

### 问题4：Git提交失败
**现象**：`Author identity unknown`  
**原因**：Git未配置用户信息  
**解决**：运行 `git config user.email` 和 `git config user.name` 配置用户信息
